# Contributing

Read `AGENTS.md` before working. RecompHamr 2.0 is currently under a feature-parity freeze.

## Required Workflow

1. Start with a goal packet.
2. Refresh memory by reading the required docs in `AGENTS.md`.
3. Update behavior, docs, help, docstrings, parity rows, and tests together.
4. Run `make verify`.
5. End with Changed, Documented, Verified, Coverage, Security, and Known limits.

Do not submit placeholders, TODO-only files, fake success paths, speculative APIs, or undocumented behavior.
